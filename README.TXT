# Learner Data Processor

This JavaScript program processes learner submission data to calculate their average scores and scores for each assignment. It follows specific rules for calculating scores, handling late submissions, and error conditions.

## Features

- Processes learner submission data for a given course and assignment group.
- Calculates each learner's average score based on their submissions.
- Considers late submissions and adjusts scores accordingly.
- Handles potential errors in the input data gracefully.
- Outputs processed data in the specified format.

## Requirements

- Node.js installed on your machine to run the code.
- Basic understanding of JavaScript programming language.

## Installation

1. Clone this repository to your local machine:

   ```
   git clone <repository-url>
   ```

2. Navigate to the project directory:

   ```
   cd learner-data-processor
   ```

3. Install dependencies:

   ```
   npm install
   ```

## Usage

1. Modify the `CourseInfo`, `AssignmentGroup`, and `LearnerSubmissions` objects in the `index.js` file to match your data.
2. Run the program:

   ```
   node index.js
   ```

3. View the processed learner data in the console output.
